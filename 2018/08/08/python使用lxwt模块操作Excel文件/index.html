enumerate（）函数：

enumerate（）是python的内置函数，适用于python2.x和python3.x
enumerate在字典上是枚举、列举的意思
enumerate参数为可遍历/可迭代的对象（如列表，字符串）
enumerate多用于在对于循环中得到计数，利用它可以同时获得索引和值，即需要索引和值值的时候可以使用
